Introduction to WildStar Addons

A Wildstar Addon consists of

  • A TOC file (toc.xml) - an XML file listing the script and form files included in the addon
  • Lua script file(s) (.lua) - Lua scripts control how the addon function
  • [optional] Form file(s) (.xml) - Layout of the window controls that appear in the addon
  • [optional] Sprite file(s) (.xml) - Additional 2D arts being used in the forms and window controls
  • [optional] Template file(s) (.xml) - Sets of style the forms can use to control and standardize the look of Windows

User created addons are located in %APPDATA%\NCSoft\WildStar\Addons

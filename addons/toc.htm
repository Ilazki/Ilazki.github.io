The TOC File

  • toc.xml inside the Addon's folder
  • Lists all the Lua files and Form files to load
  • Version information can be put here as well as hints to the game on when to exclude the Addon from loading

Example:


<Addon Name="MyTestAddon" Version="1">   
	<Script Name="MyTestAddon.lua" />   
	<Form Name="MyTestAddon.xml" />
</Addon>

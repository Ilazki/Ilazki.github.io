Lua Script

  • Lua scripts contains the logic that drives the Addon.
  • Every Wildstar addon must consists of at least one Lua script.
  • The script is run when the game world is loaded.
  • May be edited with Houston's Lua Editor or with other text editors

Example:

MyTestAddon.lua 

-----------------------------------------------------------
-- Client Lua Script for MyTestAddon
-- Copyright (c) NCsoft. All rights reserved
-----------------------------------------------------------

function MyTestAddon:Init()
    Apollo.RegisterAddon(self)
end
